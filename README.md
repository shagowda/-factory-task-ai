# Factory Task AI Management System

## Overview
An offline, AI-assisted task management system for food production factories.

## Features
- Automatic task categorization (ML-based)
- Priority prediction
- Due date suggestions
- Safety rule overrides
- Offline operation (no internet needed)

## Setup

### Requirements
- Python 3.8+
- Virtual environment

### Installation
1. Create virtual environment:
   `
   python -m venv venv
   .\venv\Scripts\Activate
   `

2. Install dependencies:
   `
   pip install -r requirements.txt
   `

3. Run the system:
   `
   python main.py
   `
